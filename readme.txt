== Description ==

This plugin allows you to improve your internal link building on your site.
For more information visit - www.viatadecocktail.ro

== Installation ==

1. Upload `seo-backlinks` folder to the `/wp-content/plugins/`directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the link 'SEO Backlinks' on the left menù of the admin interface
4. Type a key and the associated url, then click 'Save Keys!' button
5. Repeat the point 4 until you're satisfied
